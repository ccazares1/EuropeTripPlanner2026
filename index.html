🌍 One-time setup

This planner uses JSONBin.io (free, no account needed for your family) to sync edits live across all devices. Takes about 60 seconds to set up.

1
Create a free JSONBin account

Go to jsonbin.io → Sign up free → go to API Keys in your dashboard → copy your Secret Access Key.

2
Create a Bin

In JSONBin, click Create Bin → paste {} as the content → Save. Copy the Bin ID from the URL (looks like 64a3f...).

3
Save & launch

Your key and bin ID are saved only in this browser. Family members who open your shared link use the same bin automatically — they never need to set anything up.
